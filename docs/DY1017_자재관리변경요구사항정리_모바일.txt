생산관리자 화면 (모바일 화면) - 자재관리 (Admin, 데스크탑 화면)

참고 : 제품 = 자재

[생산관리자 화면]
1. 생산정보 
    1.1 "자재별 월간통계" 섹션
        예> 
        - "NPC-1000" 자재 : 생산량 500, 판매량 200, 재고량 300
        - "NPC-9000" 자재 : 생산량 300, 판매량 150, 재고량 150 
    
    1.2 "생산 정보 입력" 섹션
        -제품명 선택 : 콤보박스 CustomSelect 적용 (리스트는 시스템관리자 화면에서 정의한 제품(자재) 리스트)
        -생산날짜 : 
        -수량입력 (생산수량) : +, - 기능 활용한 입력, 참고 order2_production.html(생산관리 페이지) 참고
        -주문날짜
        -수량입력 (주문수량) 
        -현장명 검색 (현장명을 입력하세요)  or 현장명 콤보박스 CustomSelect
        -거래처명 검색 (거래처명을 입력하세요) or 거래처명 콤보박스 CustomSelect  (이때, 거래처=소속사=파트너사)
        -청구방식 (즉시청구 or 월말청구 중1택) : 콤보박스 CustomSelect
        -배송방식 (택배, 화물 or 착불 중1택) : 콤보박스 CustomSelect
        -선불/착불 (선불 or 착불 중1택) : 콤보박스 CustomSelect
        -영수증 첨부 : "영수증 업로드" 버튼 클릭 시, 파일 선택 & 업로드 기능 작동 (웹브라우저 네이트브 기능 활용)
        -메모 (텍스트 입력) : "요청사항을 입력하세요" 메모 텍스트 입력하면
        -"취소하기"버튼 & "저장하기" 버튼

    1.3 "생산 이력 조회" 섹션 : 위 "생산 정보 입력"한 리스트를 보여주는 섹션
        -검색 기능
        -생산 이력 : 생산 정보 입력 내용 이력 리스트 보기 (표) : 일자별 생산량/판매량(출고량)/재고량 한눈 볼 수 있는 표 
        -'생산일자' 선택 또는 '상세'버튼 클릭 시,'생산 상세 정보' 페이지로 이동하여 생산이력 상세 정보 제공하기


2. 출고배송 
    2.1 이번달 출고
        - 출고건수
        - 출고수량
        - 출고금액
    2.2 검색 섹션
        - 자재 필터 : 전체 or 자자명들 - "전체"버튼, "자재명1"버튼, "자재명2"버튼 ..."자재명n"버튼
        - 현장명,거래처명 검색 창 : 검색어 창
        - 년월 선택 : 년월 선택 (order2_production.html 파일 참조)
        - 상태 선택 : 전체 또는 완료, 대기 중1택, 콤보박스 CustomSelect
            > "완료" 상태 : 이미 배송 출고를 한 상태
            > "대기" 상태 : 생산은 해 놓고, 배송출고를 하기 전 상태
        
    2.3 출고 리스트 : 카드 형태로 보여주기
        - 거래처명 (카드제목) 
        - 현장명
        - 상태 토글 (완료/대기)
        - 선불/착불 방식의 배지 badge 표시 : 선불, 착불 등 - 시스템관리자 화면 > 자재관리 > 설정 > 배송결제방식 관리  화면, 기능, 데이터베이스 스키마 및 API 참조
        - 배송방식 배지 bage 표시 : 화물, 택배, 직접 등 - 시스템관리자 화면 > 자재관리 > 설정 > 배송결제방식 관리  화면, 기능, 데이터베이스 스키마 및 API 참조
        - 금액
        - 전자세금계산서 : 체크박스
        - 거래명세서 : 체크박스
        - 운임비 지불 : 체크박스
        - 금액 청구 : 체크박스
        - 수정 버튼 > 클릭 시, 수정 페이지로 이동 (등록 페이지와 동일한 레이아웃이어야 하고, 기존 등록 된 정보 불러와서 보여주기, 수정 후 저장 버튼 클릯 시 수정 적용)
        - 삭제 버튼 

    2.4 출고 등록 버튼 > 출고등록 페이지 : '출고등록'버튼을 클릭하면 출고등록 페이지로 이동
        - 거래처명 선택 : 콤보박스 CustomSelect
        - 현장명 선택 : 콤보박스 CustomSelect
        - 자재 선택 : 콤보박스 CustomSelect
        - 상태 선택 : 완료 or 대기 
        - 금액 : 숫자 입력 (1000 구분자 , 포함)
        - 선불/착불 방식 선택 : 시스템관리자 화면 > 자재관리 > 설정 > 배송결제방식 관리  화면, 기능, 데이터베이스 스키마 및 API 참조
        - 배송방식 선택 : 시스템관리자 화면 > 자재관리 > 설정 > 배송결제방식 관리  화면, 기능, 데이터베이스 스키마 및 API 참조
        - 전자세금계산서 체크박스 표시하기
        - 거래명세서 체크박스 표시하기
        - 운임비 지불 체크박스 표시하기
        - 금액 청구 체크박스 표시하기

3. 입고요청 : 작업자, 현장관리자, 거래처에서 입고 요청한 내용의 내역을 보여주는 화면 (기존 모바일 화면 및 데이터베이스 스키마 참조)
    3.1 대시보드 : 입고요청 관련한 통계카드형 대시보드 
    3.2 검색기능 
        - 현장명, 거래처명 검색어 창
        - 기간 (년월)
        - 자재선택 : 콤보박스 CustomSelect
    3.3 입고요청 리스트뷰
        - 일자
        - 거래처명
        - 현장
        - 수량
        - 요청자명
        - 메모 내용
