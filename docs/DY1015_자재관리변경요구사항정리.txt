자재관리 기능과 화면에 대한 요구사항 변경사항이 있음. 


자재관리 대상 요구사항 변화. 
기존에 만들어진 기능은 1개의 자재(NPC-1000)만 관리하는 것으로 요구사항이 정의되어 있었음. 
하지만, 이제는 여러 종류의 자재관리가 필요하다는 요구사항이 있음. 
따라서 시스템관리자 화면에서 관리하고자 하는 자재에 대해서 CRUD 할 수 있는 기능이 필요해. 
더불어, 기존의 모바일 사용자인 작업자, 현장관리자 역할 계정의 모바일 화면에서도 그에 맞게 자재관리 화면이 수정 개선 되어야 함. 
단, 소속사(파트너사) 역할 계정의 모바일 화면에서는 자재관리를 보는 기능 자체가 없어야 함.

1.입고요청 관리(= 주문현황)
2.생산정보 관리(‘품질상태’ 대신 특이사항/메모 위주)
3.출고배송결제 관리(= 출고·배송 정보 + 결제/수금 정보)
4.설정: 기초마스터(품목, 단위, 운송방식, 결제방식/세율, 운임정책, 거래처 + 생산관리 설정(품목관리외..)




[생산관리자(모바일)] for 별도의 역할 계정 production@inopnc.com / password123 : 별도 화면(톤앤매너 유지), 별도 소스 파일
주요 메뉴 
1.입고요청 조회

즉, 

시스템관리자 화면 admin@inopnc.com 화면의 '자재 관리' 메뉴에는 4가지 탭이 있습니다. 
1. 현장별 재고현항 : 작업자, 현장관리자가 작업지시서에 작성한 내용 그대로를 여기서 현장별로 확인할 수 있도록 구현했습니다. 
2. 입고 요청관리 : 작업자, 현장관리자가 재고관리 화면을 통해서 입고요청한 내역을 볼 수 있는 화면입니다. 
3. 생산정보 관리 : 생산관리자가 생산한 내용(품목, 수량, 생산일, 품질상태) 정보를 볼 수 있는 화면입니다. 생산관리자 앱에서 입력하면 이 화면을 통해서 본사 관리자가 확인할 수 있는 화면입니다. 
4. 출고배송 관리 : 현장별로 출고 된 자재의 상태, 출고일, 배송방식 수량 등을 파악할 수 있는 부분으로 구성하였습니다. 
--------
입고요청 관리 = 주문현황 페이지 
출고배송 관리 > 출고배송결제 관리  =  출고배송정보 + 결제정보(이 정보를 admin 화면에 보강하겠습니다.)
설정 : 설정 기능은 admin 에서 필요할 것 같습니다. 생산관리 설정 기능은 본사 관리자가 보유하고 있는게 맞지 않을까요?
