# DY1115 추가 안내 메시지 기획

## 1. 목적

도면마킹 및 현장공유함 워크플로우에서 사용자 혼선을 줄이고, 새로 도입된 스냅샷/PDF 기능을 적극 활용하도록 안내 메시지를 정비한다.

## 2. 주요 안내 포인트

1. **마킹 저장 직후**: "PDF로 저장되었습니다. 현장공유함 > 진행도면에서 바로 확인하고 공유할 수 있어요."
2. **현장공유함 PDF 버튼**: 툴팁/설명에 "PDF는 마킹 도구에서 다운받을 수 있는 공식 문서입니다." 등 안내.
3. **모바일 작업일지**: 연결된 도면 카드에 "PDF 확인" 버튼 또는 "마킹도구로 이동" 안내 문구 추가.
4. **관리자 Markup Tool**: 저장 후 Toast에 "PDF까지 보관됨" 메시지와 현장공유함 이동 버튼.
5. **현장공유함 저장/공유**: 저장/공유 모달에 "PDF 우선" 안내(이미지로 대체 되는 경우 메시지 표시).
