## 🚨 발견된 주요 이슈

### 1. 색상 토큰 불일치

**문제:** HTML 요구사항과 React 구현체 간 색상 차이

- `--bg`: #f5f7fb ≠ #F6F9FF
- `--text`: #101828 ≠ #1A1A1A

**권장사항:** tokens.css의 색상값을 HTML 요구사항에 맞춰 수정

### 2. 누락된 디자인 토큰

**문제:** 일부 CSS 변수 누락

- `--surface-2`, `--ink`, `--tag-blue-20` 등

**권장사항:** 완전한 디자인 토큰 세트 구현

### 3. 하드코딩된 색상값

**문제:** NoticeSection에서 인라인 스타일 사용

```tsx
style={{ color: '#101828', opacity: 1 }}
```

**권장사항:** CSS 변수 활용으로 변경
